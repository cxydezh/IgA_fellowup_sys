# IgA肾病随访系统 - 启动说明

## 问题修复

### 问题：多个Python进程冲突
- **原因**：有多个Python进程在运行，导致端口冲突
- **解决**：使用端口5001运行Flask应用，避免与端口5000上的其他服务冲突

## 启动方法

### 方法1：使用启动脚本（推荐）
```bash
python start_app.py
```

### 方法2：直接运行
```bash
python app.py
```

## 访问地址

- **本地访问**：http://localhost:5001
- **默认登录**：
  - 用户名：`admin`
  - 密码：`admin123`

## 测试

使用Playwright测试应用：
```bash
python test_playwright_fixed.py
```

## 注意事项

1. 如果端口5001被占用，请修改`start_app.py`中的端口号
2. 确保没有其他Python进程在运行，避免冲突
3. 首次运行会自动创建数据库和示例数据
